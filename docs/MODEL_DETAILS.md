# 模型详细文档

## 自训练模型 (多任务语音分析)

### 架构
```
音频输入 (16kHz)
    ↓
Mel Spectrogram (80-dim)
    ↓
频谱学习CNN + 三维注意力 (CBAM)
    ↓
├─→ 说话人嵌入头 → 192-dim embedding
├─→ 年龄回归头 → 年龄估计
├─→ 性别分类头 → 性别预测
└─→ 情绪分类头 → 情绪识别
```

### 骨干网络
- **类型**: 频谱学习CNN + CBAM注意力
- **参数量**: ~5M
- **输入**: Mel Spectrogram (batch, 1, 80, time)
- **输出**: (batch, 256, 1, time)

### 任务头
| 任务 | 输出维度 | 损失函数 | 权重 |
|------|---------|---------|------|
| 说话人识别 | 192-dim + N_speakers | CrossEntropy + Triplet | 0.8 |
| 年龄估计 | 5类 + 1回归值 | CrossEntropy + MSE | 0.3+0.3 |
| 性别识别 | 2类 | CrossEntropy | 0.5 |
| 情绪识别 | 7类 | CrossEntropy | 1.0 |

### 训练结果
| 指标 | 训练集 | 验证集 |
|------|--------|--------|
| Loss | 6.3411 | 7.0507 |
| 说话人准确率 | 69.1% | 0.0%* |
| 性别准确率 | 95.8% | 100.0% |
| 情绪准确率 | 41.2% | 39.4% |

*验证集说话人不在训练集中(数据集划分问题)

### 量化模型
| 格式 | 大小 | 情绪准确率 | 性别准确率 |
