# 模型架构设计文档

## 概述

多任务语音分析模型，同时实现说话人识别、年龄估计、性别分类和情绪识别。

## 关键设计决策

### 1. 骨干网络选择

| 方案 | 参数量 | 优点 | 缺点 | 选择 |
|------|--------|------|------|------|
| **频谱学习CNN+注意力** | ~5M | CPU友好，频谱特征强 | 序列建模弱 | ✅ |
| Transformer | ~20M | 全局建模强 | 端侧延迟高 | ❌ |
| ECAPA-TDNN | ~6M | 说话人强 | 情绪弱 | ❌ |
| Wav2Vec 2.0 | ~95M | 通用性强 | 太大 | ❌ |

### 2. 注意力机制

采用CBAM (Convolutional Block Attention Module):
- **通道注意力**: 学习"哪些频率重要"
- **空间注意力**: 学习"哪些时间段重要"

### 3. 多任务头设计

| 任务 | 输出 | 损失函数 | 权重 |
|------|------|----------|------|
| 说话人 | 192-dim embedding | AAMSoftmax | 0.8 |
| 情绪 | 7类 | CrossEntropy | 1.0 |
| 年龄 | 5类 + 回归值 | CE + MSE | 0.3+0.3 |
| 性别 | 2类 | CrossEntropy | 0.5 |

### 4. 输入表示

- **Mel Spectrogram**: 80维，更符合人耳感知
- **帧移**: 256 samples (16ms @ 16kHz)
- **输入长度**: 300帧 (~4.8s)，支持变长

## 模型规格

| 指标 | 标准版 | 轻量版 |
|------|--------|--------|
| 参数量 | ~8M | ~3M |
| FP32大小 | ~32MB | ~12MB |
| INT8大小 | ~8MB | ~3MB |
| 骨干通道 | [32,64,128,256] | [16,32,64,128] |
| 残差块数 | 2/阶段 | 1/阶段 |
| 推理延迟 | ~200ms | ~100ms |

## 端侧适配

### MT9655约束
- **CPU**: ARM Cortex-A55 (4核)
- **内存**: 2-4GB
- **NeuroPilot**: 支持TFLite/ONNX

### 优化策略
1. **INT8量化**: 减少4x模型大小
2. **频率池化**: 减少序列长度
3. **轻量骨干**: 减少50%参数量
4. **XNNPack**: 加速CPU推理

## 训练策略

### 阶段1: 单任务预训练
- 先用各任务单独训练骨干
- 冻结骨干，训练任务头

### 阶段2: 多任务联合训练
- 解冻所有参数
- 联合损失优化
- 动态损失权重调整

### 阶段3: 微调
- 针对特定数据集微调
- 数据增强增强泛化
